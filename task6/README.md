# Task 6 - Sales Trend Analysis Using Aggregations

## Objective

Analyze monthly revenue and order volume using SQL aggregation functions on an online sales dataset.

---

## Tools Used

- SQLite
- DB Browser for SQLite
- SQL
- Visual Studio Code

---

## Dataset

- online_sales.csv

---

## Analysis Performed

- Imported the online sales dataset into SQLite.
- Displayed all records from the table.
- Calculated monthly revenue using `SUM()`.
- Calculated monthly order volume using `COUNT(DISTINCT order_id)`.
- Identified the top 3 months by total sales.
- Calculated revenue generated by each product.
- Calculated average monthly revenue.
- Sorted results using `ORDER BY`.
- Grouped data using `GROUP BY`.
