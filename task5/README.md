# Task 5 - Exploratory Data Analysis (EDA)

## Objective

Perform Exploratory Data Analysis (EDA) on the Titanic dataset to understand data patterns, identify relationships, detect missing values, and extract meaningful insights using statistical analysis and data visualization.

---

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset

- Titanic Dataset (CSV)

---

## Analysis Performed

- Loaded the Titanic dataset using Pandas.
- Displayed the first five records using `head()`.
- Examined the dataset structure using `info()`.
- Generated statistical summaries using `describe()`.
- Analyzed categorical data using `value_counts()`.
- Created a histogram to visualize the distribution of passenger ages.
- Used a boxplot to identify outliers in the Fare column.
- Created a scatter plot to analyze the relationship between Age and Fare.
- Generated a correlation heatmap for numerical features.
- Created a pairplot to explore relationships between important variables.
- Documented observations for each visualization.
- Summarized the overall findings from the analysis.

---

## Visualizations

- Histogram
- Boxplot
- Scatter Plot
- Correlation Heatmap
- Pairplot

---

## Key Findings

- The dataset contains 891 passenger records and 12 features.
- Missing values were identified in the Age, Cabin, and Embarked columns.
- Most passengers were young adults.
- Male passengers outnumbered female passengers.
- The Fare column contains several high-value outliers.
- Most numerical variables show weak correlations.
- Higher-class passengers and those who paid higher fares generally had better survival rates.
