# Task 3 - SQL for Data Analysis

## Objective
Use SQL queries to extract, manipulate, and analyze data from a relational database.

## Tool Used
- MySQL Workbench 8.0

## Database
- internship_db

## Tables Created
- employees
- departments

## SQL Concepts Covered
- CREATE DATABASE
- CREATE TABLE
- INSERT INTO
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- Aggregate Functions (SUM, AVG)
- Subqueries
- CREATE VIEW
- CREATE INDEX

## Queries Performed
- Created a new database.
- Created `employees` and `departments` tables.
- Inserted sample records into both tables.
- Retrieved data using SELECT statements.
- Filtered records using the WHERE clause.
- Sorted records using ORDER BY.
- Grouped records using GROUP BY.
- Performed INNER, LEFT, and RIGHT JOIN operations.
- Calculated total and average salary using aggregate functions.
- Executed a subquery to filter employees based on average salary.
- Created a SQL view for simplified data analysis.
- Created an index to improve query performance.
